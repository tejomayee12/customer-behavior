## Overview

This project demonstrates an end-to-end **data analytics workflow** focused on converting raw data into **actionable business insights**. It covers data ingestion, exploratory analysis, data cleaning, SQL-based analysis, and interactive dashboard development. The final outputs support **KPI tracking, pricing analysis, risk identification, and decision-making** for stakeholders.

The project is designed to reflect real-world analytics tasks commonly performed by **Data Analysts** in business environments.



## Dataset

* Source: Public / Synthetic business dataset (sales / operations / customer data)
* Format: CSV
* Size: Medium-scale (suitable for SQL + BI analysis)
* Key Attributes:

  * Transaction details
  * Customer and product information
  * Revenue, cost, and pricing variables
  



## Tools & Technologies

* **Python**: Data loading, EDA, data cleaning

  * Libraries: pandas, numpy, matplotlib, seaborn
* **SQL**: Data querying and aggregation

  * Databases: PostgreSQL / MySQL / SQL Server
* **Power BI**: Interactive dashboard development
* **Gamma**: Report and presentation (PPT) creation
* **Git & GitHub**: Version control and project documentation


## Project Workflow

1. **Data Loading**

   * Imported raw dataset into Python using pandas
   * Performed initial structure and schema checks

2. **Exploratory Data Analysis (EDA)**

   * Analyzed data distribution, trends, and outliers
   * Identified missing values and data quality issues
   * Examined relationships impacting revenue and KPIs

3. **Data Cleaning & Preparation**

   * Handled missing and inconsistent values
   * Standardized data types and formats
   * Created derived metrics for analysis

4. **SQL Analysis**

   * Loaded cleaned data into relational databases
   * Wrote optimized SQL queries for:

     * KPI calculations
     * Pricing and revenue analysis
     * Risk and anomaly identification
     * Trend and performance reporting

5. **Dashboard Development (Power BI)**

   * Designed an interactive dashboard for stakeholders
   * Included slicers, filters, and drill-down views
   * Focused on clarity, usability, and business impact

6. **Reporting & Presentation**

   * Created a concise analytical report summarizing findings
   * Built a professional PPT using Gamma for storytelling
